全局字体更改

即然用了这么geek的主题，改成mono字体看起来的效果会更舒服，此处我改成了Menlo字体。

在themes/github-style/static/css/github-style.css中的body内添加font-family

body {
  margin: 0;
  font-family:  Menlo, "Powerline Consolas", "PT Mono";
}

个人简介部分的字体需要在themes/github-style/layouts/partials/overview.html中更改。定位以下位置，添加style=内容

<article class="markdown-body entry-content container-lg f5" style="font-family: Menlo;">
    {{- .Content -}}
</article>

标题打字机效果

在themes/github-style/static/css/github-style.css中增加打字机效果

.wrapper {
  height: auto;
  /*This part is important for centering*/
  display: grid;
  place-items: left;
}
.typing-demo {
  width: 22ch;
  animation: typing 2s steps(22), blink .5s step-end infinite alternate;
  white-space: nowrap;
  overflow: hidden;
  border-right: 3px solid;
  font-family: Menlo, monospace;
  /* font-size: 2em; */
}

@keyframes typing {
  from {
    width: 0
  }
}
    
@keyframes blink {
  50% {
    border-color: transparent
  }
}
